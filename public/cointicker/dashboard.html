🔄 데이터 흐름 (Data Flow)

  1. 수집: 5개 데이터 소스에서 5~30분 주기로 크롤링
  2. 검증 & 정제: 스키마 검증, 결측치 처리, 이상치 제거
  3. 지표 계산: 8가지 기술적 지표 + 감성 분석 (FinBERT)
  4. 저장: HDFS (원본) + MariaDB (처리 데이터) + Redis (캐시)
  5. 분석 & 알림: 심각도별 알림 생성 (Critical/High/Medium/Low)
  6. 시각화: WebSocket을 통해 대시보드에 실시간 푸시
  7. 알림 전송: 이메일, Slack, 푸시 알림 다중 채널

🎨 UI/UX 설계 원칙

1️⃣ 정보 밀도

한 화면에 모든 핵심 지표를 배치하여 사용자가 여러 페이지를 넘기지 않고도 시장 전체를 파악할 수 있도록 설계

2️⃣ 색상 코드

긍정(초록), 부정(빨강), 중립(회색) 등 직관적인 색상으로 빠른 인지 가능. 심각도별 알림도 색상으로 구분

3️⃣ 실시간 업데이트

WebSocket 연결로 1초 단위 업데이트. 새로운 알림 발생 시 푸시 알림 + 시각적 효과

4️⃣ 반응형 디자인

모바일, 태블릿, 데스크톱 모든 기기에서 최적화된 레이아웃. Grid 시스템으로 자동 조정

5️⃣ 맞춤 설정

사용자가 관심 있는 코인, 지표, 알림 조건을 자유롭게 설정. 개인화된 대시보드 구성

6️⃣ 데이터 내보내기

CSV, JSON, PDF 형식으로 데이터 내보내기 가능. 자체 분석 및 백테스팅 지원

🚀 핵심 기능

✅ 실시간 모니터링

  • WebSocket 연결로 1초 단위 가격 업데이트
  • 기술적 지표 5분 주기 재계산
  • 뉴스 감성 분석 15분 주기 업데이트

🔔 스마트 알림 시스템

  • 심각도별 필터링 (Critical/High/Medium/Low)
  • 중복 알림 방지 (쿨다운 10분)
  • 다중 채널 알림 (이메일, Slack, 푸시, SMS)
  • 사용자 맞춤 조건 설정

📊 고급 차트

  • 캔들스틱 차트 + 거래량 연동
  • 볼린저밴드, 이동평균선 오버레이
  • 지지선/저항선 자동 표시
  • 줌/팬 기능, 타임프레임 변경 (1분 ~ 1일)

🎯 패턴 인식

  • 머리어깨형, 이중바닥 등 차트 패턴 인식
  • 거래량 급증 감지
  • 가격 이탈 알림 (저항선/지지선 돌파)
