HIGHLIGHTS
🎯

이중 수익 구조의 안정성

1단계 B2C 광고 수익으로 초기 현금 흐름 확보 후, 2단계 B2B/B2G 구독 모델로 전환하여 예측 가능한 반복 수익(MRR) 구조 구축. 수익 다각화를 통한 리스크 분산 효과.

📈

입증된 시장 수요

대학 동아리 시장은 국내 약 40만명 이상의 TAM(Total Addressable Market)을 보유. 기존 관리 도구의 부재로 인한 명확한 Pain Point 존재. 초기 수용자 확보 용이.

💰

낮은 초기 투자 & 빠른 PMF

초기 투자금 1,064만원으로 MVP 출시 가능. 운영비 월 222만원으로 린 스타트업 방식 운영. 12-15개월 내 손익분기점 도달로 빠른 자생력 확보 예상.

🔄

높은 전환율과 재사용성

동아리 관리는 연중 지속적 니즈. B2C에서 검증된 사용자 기반을 B2B/B2G로 업셀 가능. 대학 공식 솔루션 채택 시 장기 계약(3-5년) 기대.

⚡

확장 가능한 비즈니스 모델

동아리 외 학생회, 소모임, 동문회 등으로 수평 확장 가능. 기능 고도화를 통한 기업 팀 협업 툴로의 수직 확장 잠재력 보유. API 제공 시 에코시스템 구축 가능.
