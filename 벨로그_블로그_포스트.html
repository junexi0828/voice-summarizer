AI 음성 텍스트 정리 도구 제작기: React + Vercel로 만든 현대적 웹앱

안녕하세요! 오늘은 제가 개발한 "AI 음성 텍스트 정리 도구"의 전체 제작 과정을 공유하려고 합니다. React 기반으로 개발하여 Vercel에 배포했으며, SEO 최적화까지 완료한 프로젝트의 모든 이야기를 담았습니다.

📋 프로젝트 개요

"AI 음성 텍스트 정리 도구"는 음성을 텍스트로 변환하고 다양한 AI 서비스로 자동 요약해주는 웹 애플리케이션입니다. 사용자가 마이크로 음성을 녹음하면 실시간으로 텍스트로 변환되고, 선택한 AI 서비스(Claude, GPT, Groq, Perplexity, Gemini)를 통해 자동으로 요약해주는 서비스입니다.

🛠 기술 스택 선택

Frontend: React 18.2.0

  • 이유: 최신 기능과 안정성, 풍부한 생태계
  • 장점: 컴포넌트 기반 개발, 빠른 개발 속도
  • 결과: 유지보수 용이한 코드 구조

Styling: Tailwind CSS v3.4.17

  • 이유: 유틸리티 퍼스트 접근법, 빠른 스타일링
  • 도전: v3 설치 시 호환성 문제 발생
  • 해결: 정확한 버전 매칭으로 해결

배포: Vercel

  • 이유: React 생태계 최적화, 자동 배포
  • 장점: Git 연동, 무료 SSL, 빠른 CDN
  • 결과: 도메인 연결까지 완벽한 배포 환경

🎯 핵심 기능 개발 과정

1. 음성 인식 시스템

Web Speech API를 활용하여 실시간 음성 인식을 구현했습니다.

// Web Speech API 활용
const SpeechRecognition = window.SpeechRecognition || window.webkitSpeechRecognition;
recognition.continuous = true;
recognition.interimResults = false; // 정확도 향상
도전과 해결:
  • 실시간 스트리밍 문제: 중간 결과를 무시하여 정확도 향상
  • 브라우저 호환성: Chrome, Edge, Safari 지원 확인
  • 언어 설정: 한국어/영어 전환 기능 구현

2. AI 서비스 연동

다양한 AI 서비스를 지원하여 사용자가 선택할 수 있도록 했습니다:

  • Claude (Anthropic): sk-ant- 형식 API 키
  • GPT (OpenAI): sk- 형식 API 키
  • Groq: gsk_ 형식 API 키
  • Perplexity: pplx- 형식 API 키
  • Gemini (Google): Google AI Studio API 키
보안 고려사항:
  • API 키는 브라우저 로컬에만 저장
  • 서버로 전송되지 않음
  • HTTPS 통신으로 암호화

3. 사용자 경험 개선

  • 자동 높이 조절: react-textarea-autosize 라이브러리
  • 복사 기능: 클립보드 API 활용
  • 반응형 디자인: 모바일 최적화
  • 에러 처리: 구체적인 오류 메시지

⚡ 개발 과정의 주요 도전과 해결

1. Tailwind CSS v3 설치 문제

문제: 버전 충돌로 인한 설치 실패 
# 문제: 버전 충돌로 인한 설치 실패
npm install tailwindcss@^3.4.17

# 해결: 정확한 버전 매칭
npm install tailwindcss@3.4.17 autoprefixer@^10.4.21 postcss@^8.5.6

2. 모바일 반응형 최적화

문제: PC에서도 작은 버튼이 표시됨
해결: 적절한 브레이크포인트 사용 
/* 문제: PC에서도 작은 버튼이 표시됨 */
.sm:px-6 → .md:px-6

/* 해결: 적절한 브레이크포인트 사용 */
.md:px-6 /* 768px 이상에서만 적용 */

3. SEO 최적화

  • 구조화 데이터: JSON-LD 스키마 추가
  • 메타 태그: Open Graph, Twitter Card 설정
  • Sitemap: XML 사이트맵 생성
  • Robots.txt: 검색엔진 크롤링 허용

Vercel 배포 과정

  • GitHub 연동: 자동 배포 설정

도메인 설정

무한 리디렉션 문제를 해결하기 위한 vercel.json 설정:

// vercel.json (무한 리디렉션 문제 해결)
{
  "redirects": [
    {
      "source": "/",
      "has": [{"type": "host", "value": "www.eieconcierge.com"}],
      "destination": "https://eieconcierge.com",
      "statusCode": 301
    }
  ]
}
